# Mood-Quote-Generator
A simple mood-based quote generator that provides personalized motivational quotes based on user feelings.

## Features

- User enters their name
- User enters their mood (sad, angry, tired, etc.)
- The app generates a motivational quote
- Quote includes the user's name for personalization
- Responsive design using Bootstrap

## Technologies Used

- HTML
- CSS
- JavaScript
- jQuery
- Bootstrap

## How it Works

1. User enters their name.
2. User types their current mood.
3. The application detects the mood using JavaScript logic.
4. A personalized quote is generated and displayed.

Example:

Input:
Name: Vineela  
Mood: I am sad

Output:
"Vineela, tough times never last but strong people do."
